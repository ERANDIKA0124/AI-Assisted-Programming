Version A - Traditional Java Implementation

This version is written as a simple Java console application in one main file. It represents the traditional approach where the developer writes the logic manually and improves it step by step.

Features

- Add student records
- View all student records
- Search by student ID
- Update assignment and exam marks
- Delete student records
- Calculate class average, highest average, and lowest average
- Save/load records using a CSV file

How to Run

Open a terminal inside the `src` folder and run:

```bash
javac StudentPerformanceTraditional.java
java StudentPerformanceTraditional
```
