Version B - AI-Assisted Java Implementation

This version represents the AI-assisted approach. The idea is not to blindly copy code. The developer first understands the problem, creates a simple manual version, and then uses AI suggestions to improve code structure, validation, and maintainability.

Main improvements compared with Version A

- Separate classes for model, service, repository, input, and application logic
- Better input validation
- Better error messages
- Cleaner method responsibilities
- Use of `Optional` for safe searching
- Separate summary object for statistics

How to Run

Open a terminal inside the folder and run:

```bash
javac *.java
java App
```
