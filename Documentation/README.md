Project Documentation

Project Title

Student Performance Management System using Java

Project Purpose
The purpose of this system is to manage student marks and calculate simple performance measures. This project is suitable for the assessment because it allows comparison between manual programming and AI-assisted programming using the same problem.

Main Functions
Add student details
View student records
Search student by ID
Update marks
Delete student record
Calculate weighted average and grade
Show class summary statistics
Save and load records from CSV file

Why this topic was selected

As a BSc Industrial Statistics and Mathematical Finance undergraduate, I selected a student performance system because it connects programming with data handling and basic statistical analysis. The system is not too complex, but it is enough to show Java classes, arrays/lists, file handling, input validation, and simple calculations.

Technologies Used

*Java

*Java Collections Framework

*File handling using CSV

*Console-based user interface

*UML diagrams
