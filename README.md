- Version B: AI-assisted Java implementation

Repository Contents

- Version_A_Traditional - Simple single-file Java implementation
- Version_B_AI_Assisted - Modular Java implementation with separate classes

## Java run commands

Version A

```bash
cd Version_A_Traditional/src
javac StudentPerformanceTraditional.java
java StudentPerformanceTraditional
```

Version B

```bash
cd Version_B_AI_Assisted/src
javac *.java
java App
```
